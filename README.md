# Carbon Footprint Tracker 

A web-based application that helps users track and understand their daily carbon
emissions based on transportation and electricity usage.

## Problem Statement
People are often unaware of their daily carbon emissions caused by routine
activities, and there is no simple way to track or reduce them.

## Solution
This application calculates a user's carbon footprint based on daily activities
and presents the results through a simple, user-friendly dashboard.

## Features
- User authentication (Login / Signup)
- Carbon footprint calculation
- Dashboard showing total emissions
- History of carbon usage
- Cloud-based data storage using Firebase

## Technologies Used
- HTML, CSS, JavaScript
- Firebase Authentication
- Firebase Firestore
- Firebase Hosting

## Architecture
The frontend collects user inputs and sends data to Firebase services where
authentication, storage, and deployment are handled.

## Future Enhancements
- AI-based emission prediction
- Personalized reduction tips
- Mobile application
